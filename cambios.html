Apartado 1: Nuevas etiquetas añadidas respecto a la práctica anterior. Si ya estaban añadidas, indicar qué cambios se han realizado.

  • Etiquetas semánticas nuevas:
    • Se ha añadido la etiqueta <article> para dividir contenido relacionado dentro de secciones específicas
    • Incorporación de <aside> en varias páginas para agregar información adicional, como enlaces a recursos externos.
    • Uso de <figure> y <figcaption> para mejorar la presentación de imágenes con descripciones en secciones multimedia.
    • Se ha implementado <section> en todas las páginas para mejorar la organización del contenido.
  • Mejoras en navegación y accesibilidad:
    • Se ha añadido la etiqueta <nav> para estructurar el sistema de navegación con menús desplegables.
    • Incorporación de atributos ARIA para mejorar la accesibilidad.
  • Tablas
    • Se han añadido etiquetas <table>, <tr>, <td> con nuevas filas y columnas para una mejor presentación de datos.
    • Uso de la clase table-responsive de Bootstrap para hacer las tablas adaptables a dispositivos móviles.

Apartado 4: Indicad en qué páginas habéis incluido las media queries creadas y qué conseguís con las mismas.

Se aplican media queries en la mayoría de las páginas con el objetivo de ajustar los tamaños de texto e imágenes a pantallas más pequeñas. Un ejemplo de la que he usado es:

                    
                                    @media (max-width: 768px) {
                                        nav {
                                            width: 100%;
                                            display: flex;
                                            flex-direction: column;
                                        }
                                    }
