Introducción.

La programación suele ser la asignatura clave en los primeros semestres de este ciclo formativo. El objetivo de esta asignatura es proporcionar los fundamentos de la programación necesarios para desarrollar aplicaciones informáticas, tanto para dispositivos móviles como para entornos de escritorio.

Ideas principales.

  • Conceptos básicos: Qué es la programación, el ciclo de desarrollo de software, y los tipos de programas.
  • Lenguajes de programación: Diferencia entre lenguajes de bajo y alto nivel. Introducción a los lenguajes de programación más comunes (como Python, C, Java, JavaScript, etc.).
  • Compiladores e intérpretes: Explicación del proceso de compilación y ejecución de un programa.
  • Entornos de desarrollo (IDE): Herramientas y programas que facilitan la escritura, depuración y ejecución de código (por ejemplo, Visual Studio Code, PyCharm, Eclipse).

Sintaxis y estructuras básicas.

  • Sintaxis básica: Declaración de variables, tipos de datos básicos (enteros, flotantes, cadenas, booleanos).
  • Operadores: Operadores aritméticos, lógicos, de comparación, y de asignación.
  • Estructuras de control:
    • Condicionales: Uso de if, else, elif (en Python) para la toma de decisiones.
    • Bucles: Uso de bucles for, while y do-while para repetir operaciones.
    • Sentencias de control: Uso de break, continue y return.

Funciones y procedimientos.

  • Definición de funciones: Cómo crear y usar funciones para modularizar el código.
  • Parámetros y argumentos: Cómo pasar datos a las funciones (por valor y por referencia).
  • Valor de retorno: Uso de la palabra clave return para devolver resultados de una función.
  • Funciones recursivas: Qué es la recursión y cómo usarla en problemas como el cálculo de factoriales, Fibonacci, etc.

Estructuras de datos básicas.

  • Arrays y listas: Cómo declarar, inicializar y manipular arreglos o listas de elementos.
  • Cadenas de texto: Manipulación de strings, funciones comunes para concatenar, dividir y modificar cadenas.
  • Manejo de índices: Acceso y modificación de elementos dentro de estas estructuras.

Algoritmos básicos.

  • Algoritmos de búsqueda: Búsqueda secuencial, búsqueda binaria (en listas ordenadas).
  • Algoritmos de ordenación: Algoritmos como burbuja, selección, inserción, quicksort y mergesort.
  • Análisis de la complejidad: Introducción a la notación Big O para evaluar la eficiencia de los algoritmos.

Programación Orientada a Objetos.

  • Conceptos fundamentales de la POO: Clases, objetos, atributos y métodos.
  • Encapsulamiento: Cómo proteger los datos de una clase utilizando modificadores de acceso (público, privado, protegido).
  • Herencia: Uso de clases base y derivadas para crear jerarquías de clases.
  • Polimorfismo: El concepto de que una misma interfaz puede tener múltiples implementaciones.
  • Abstracción: Cómo ocultar detalles complejos y proporcionar interfaces sencillas para interactuar con objetos.

Manejo de archivos y entrada/salida.

  • Lectura y escritura de archivos: Cómo manejar archivos de texto (abrir, leer, escribir y cerrar archivos).
  • Archivos binarios: Cómo trabajar con archivos binarios y cómo serializar y deserializar objetos.
  • Entrada estándar (stdin):Cómo leer datos del usuario desde la consola.
  • Salida estándar (stdout): Cómo mostrar datos en la consola o interfaz gráfica.

Manejo de errores y excepciones.

  • Errores de compilación vs. errores en tiempo de ejecución: Diferencias entre errores que ocurren al compilar y cuando el programa ya está en ejecución.
  • Excepciones:Manejo de excepciones con estructuras como try, catch, finally.
  • Trazas y depuración: Uso de herramientas de depuración para identificar y corregir errores en el código.

Estrcuturas de datos avanzadas.

  • Pilas (Stacks): Implementación y uso de pilas, y su aplicación en tareas como deshacer/rehacer.
  • Colas (Queues): Implementación y uso de colas, y su aplicación en procesos de gestión de tareas.
  • Listas enlazadas: Concepto de listas dinámicas con nodos, y cómo insertar, eliminar y recorrer elementos.
  • Árboles y grafos: Estructuras jerárquicas y no jerárquicas, y su uso en representaciones de datos complejos.

Herramientas y tecnologías utilizadas

  • Lenguajes de programación: Python, C, C++, Java, JavaScript, etc.
  • Entornos de desarrollo: Visual Studio Code, PyCharm, Eclipse, IntelliJ IDEA.
  • Sistemas de control de versiones: Git (y plataformas como GitHub o GitLab para gestionar proyectos en equipo).
  • Herramientas de depuración: Depuradores integrados en IDEs, como los de PyCharm, Visual Studio, o GDB.
