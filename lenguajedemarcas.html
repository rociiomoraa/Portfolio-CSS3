Introducción

La asignatura de Lenguajes de Marcas tiene como objetivo proporcionar a los estudiantes los conocimientos necesarios para desarrollar y gestionar contenido web y multimedia, mediante el uso de lenguajes de marcas. Estos lenguajes son fundamentales para la creación de sitios web estáticos y dinámicos, ya que permiten estructurar y dar formato al contenido en la web.

HTML (Hypertest Markup Language)

  • Estructura básica de un documento HTML:
    • Definición de las partes principales de una página web (elementos como html, head, body, etc.).
    • La importancia de las etiquetas de apertura y cierre.
    • Concepto de árbol DOM (Document Object Model) y cómo los navegadores interpretan el código HTML.
  • Etiquetas básicas:
    • Estructura de texto: h1, p, ul, ol, li, etc.
    • Enlaces: a(hipervínculos) y cómo definir URLs.
    • Imágenes: img, atributos como src, alt, y su correcta utilización.
    • Tablas: table, tr, th, td y cómo organizarlas.
    • Formularios: form, input, select, textarea, etc. para recopilar datos de los usuarios.
  • Atributos HTML: Uso de atributos como id, class, style, src, href, alt para personalizar el comportamiento y presentación de los elementos.
  • Comentarios HTML: Cómo agregar comentarios en el código que no son visibles para los usuarios pero útiles para los desarrolladores.
  • Validación de HTML: Cómo comprobar que el código HTML es correcto utilizando herramientas como el validador W3C.

CSS (Cascading Style Sheets)

  • Fundamentos de CSS:
    • Qué es CSS y para qué se utiliza (separar la estructura del contenido de la presentación).
    • Selección de elementos HTML mediante selectores: clases, identificadores (ID), etiquetas, etc.
  • Propiedades básicas de CSS
    • Estilos de texto: color, tamaño, fuente, alineación, etc.
    • Estilos de fondo: colores de fondo, imágenes de fondo, propiedades de repetición.
    • Márgenes y rellenos: cómo manipular los márgenes (margin) y el relleno (padding) de los elementos.
    • Bordes: cómo establecer bordes alrededor de los elementos.
  • Modelo de caja (Box Model): Comprensión de cómo los márgenes, bordes, relleno y contenido forman la caja de un elemento en CSS.
  • Posicionamiento de elementos:
    • Posicionamiento estático, relativo, absoluto y fijo (position).
    • Flexbox: uso de display: flex para crear diseños flexibles y responsivos.
    • Grid: conceptos básicos de display: grid para la creación de layouts más complejos.
  • Estilos para enlaces, botones y formularios: Cómo personalizar la apariencia de los enlaces, botones y formularios en una página web.
  • Media Queries: Introducción a la adaptación de las páginas web a diferentes dispositivos y tamaños de pantalla (responsividad).

Introducción a JavaScript

  • Conceptos básicos de JavaScript: Cómo se utiliza JavaScript para hacer que una página web sea interactiva.
  • Manipulación del DOM: Cómo seleccionar y modificar elementos HTML a través de JavaScript.
  • Eventos: Cómo manejar eventos en JavaScript (como click, load, mouseover, etc.).
  • Validación de formularios: Uso de JavaScript para validar datos de formularios antes de enviarlos al servidor.

XML (Extensible Markup Language)

  • Estructura básica de XML:
    • Diferencias entre HTML y XML.
    • Cómo definir documentos XML y qué es una etiqueta de apertura y cierre.
  • Atributos en XML: Cómo definir y usar atributos dentro de las etiquetas XML.
  • Uso de XML en aplicaciones: Cómo se utiliza XML para almacenar y transmitir datos entre aplicaciones.
  • Validación de documentos XML: Introducción a DTD (Document Type Definition) y esquemas XML (XSD).

Otras tecnologías de marca

  • JSON (JavaScript Object Notation): Qué es JSON, cómo se utiliza para almacenar y transmitir datos, y su relación con XML.
  • SVG (Scalable Vector Graphics): Introducción a los gráficos vectoriales escalables, cómo crear imágenes y gráficos interactivos con código.
  • RSS (Really Simple Syndication): Uso de RSS para la distribución de contenido web (como noticias, blogs, etc.).

Buenas prácticas y estándares

  • Accesibilidad web: Cómo crear páginas web accesibles para personas con discapacidades, utilizando etiquetas y atributos adecuados.
  • SEO (Search Engine Optimization): Introducción a cómo estructurar el código HTML para que los motores de búsqueda indexen mejor el contenido.
  • Optimización de imágenes y archivos: Mejores prácticas para reducir el tamaño de las imágenes y archivos web, para mejorar el rendimiento.
  • Compatibilidad entre navegadores: Cómo asegurar que las páginas web se vean correctamente en diferentes navegadores (Chrome, Firefox, Internet Explorer, Safari, etc.).

Herramientas y recursos

    • Editores de código: Visual Studio Code, Sublime Text, Notepad++, etc.
    • Herramientas de validación: Validador W3C para HTML y CSS.
    • Plataformas de desarrollo web: CodePen, JSFiddle, o cualquier otra herramienta online para pruebas rápidas.

Objetivos

  • Crear páginas web estáticas utilizando HTML y CSS.
  • Comprender y aplicar las buenas prácticas de accesibilidad y SEO.
  • Ser capaces de crear una estructura básica de páginas web con formularios y tablas.
  • Tener conocimientos básicos de XML y JSON como lenguajes de marcas utilizados en el intercambio de datos.
